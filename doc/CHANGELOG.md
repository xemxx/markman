## v0.3.0

**本次升级修改数据结构，需服务端先升级v0.2.0及以上，不可回退，需谨慎**

**Features**

1. 支持服务器地址修改后维持账号数据（通过新增用户uuid实现，不再校验服务器地址）
2. 支持自动保存时间无极调整
3. 支持搜索笔记（包括笔记内容）
4. 更新依赖
   1. vue3
   2. vite5
   3. electron31
   4. ant-design-vue4
   5. uuid9
   6. vuex4->pinia2
5. 模板替换为 vite + ts +electron 提升开发体验

**Bugfix**

1. 反复按保存可能出现同步冲突
2. 修改后点击保存可能出现修改回滚的情况
3. 修复断网或者服务端不可用时偶现的白屏问题
